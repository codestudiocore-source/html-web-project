 HTML Web Development Projects

This repository contains three beginner-friendly HTML projects created for practice and learning purposes.  
All projects are built using pure **HTML & CSS**.

 Professional Registration Form

A fully structured HTML form using almost all input types.

Features:
- Text, Email, Password inputs
- File upload (Image)
- Radio buttons & Checkboxes
- Date, Time, Range, Color inputs
- Dropdown (Select)
- Textarea
- Submit & Reset buttons
- Professional CSS styling

Technologies Used:
- HTML5
- CSS3

 Graduation Result Sheet (100 Students Record)

A structured university-style result sheet created using HTML Tables.

Features:
- 100 student records
- Subjects marks
- Total calculation
- CGPA column
- Clean professional table design
- Zebra striping
- Hover effects
- University-style layout

Technologies Used:
- HTML (Table)
- CSS (Professional Styling)

 Full Stack Development Chapter Website

An educational chapter-style website explaining Full Stack Development.

Topics Covered:
- What is Full Stack Development
- Why it is important
- Roles in real life
- Frontend technologies
- Backend technologies
- Frameworks
- Databases
- APIs
- Development tools

HTML Concepts Used:
- Ordered Lists (`<ol>`)
- Unordered Lists (`<ul>`)
- Definition Lists (`<dl>`)
- Headings & Paragraphs
- Structured Content Layout

Technologies Used:
- HTML
- CSS (Digital Book Design)(simpe) 

Purpose of This Repository

This project was created to:

- Practice core HTML concepts
- Understand form structure
- Learn table formatting
- Apply professional CSS styling
- Build clean structured web pages
- Prepare for Full Stack Development journey
